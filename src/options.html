About Authentication

This extension uses the Google Drive API to retrieve file information. Authentication grants read-only access to your Google Drive.

How to Use

Basic Usage

1. Open a file in Google Drive or Google Docs
2. Click the Shiftly Drive icon in the browser toolbar
3. Click on the displayed parent folder links to navigate

Cache and Data

Folder information is cached for 5 minutes to reduce API requests.
